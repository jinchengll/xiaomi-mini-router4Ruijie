# xiaomi-mini-router4Ruijie
作者: jinchengll

描述: 小米路由器mini刷入固件支持锐捷校园网,实现整体流程自动化.

## 目前已知支持锐捷版本
> 如果你使用成功了,请提交你的锐捷版本给我

1. V1.65

## 使用方式
1. 安装python环境,并安装`getssh.py`文件所需要的包
2. 先将路由器刷为官方固件版本2.7.11。
3. 执行脚本`shuaji.sh`
